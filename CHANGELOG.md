# Changelog

All notable changes to CORE OS will be documented in this file.

## [0.1.0-alpha.1] - 2026-05-12

### Added
- Initialized Rust workspace with 4 crates: `host_shim`, `display_server`, `micro_kernel`, `island_mode`.
- Added CI pipeline via GitHub Actions (Rust build, test, clippy, fmt; markdown lint).
- Added project infrastructure: `README.md`, `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `SECURITY.md`, `NOTICE`, `PROJECT_STATUS.md`.
- Added issue and pull request templates.
- Added `.gitignore` and `archive/README.md` with disclaimer.
- Established bilingual documentation policy (Russian for project docs, English for code and commits).
