# CORE OS — Бизнес-модель и Go-to-Market

## Принцип: Open Source + Платные сервисы

CORE — открытая платформа. Деньги — на инфраструктуре и сервисах поверх.

---

## Модель монетизации

### 1. Hardware (Branded Hardware)
- Core-Plug / Base Station: коробочка в розетку, $49
- Неттопы для дома и офиса
- Наценка 40%
- 500 млн "продвинутых" домохозяйств. Захват 1% = $1.5 млрд выручки

### 2. B2B SaaS (Self-hosted Pro)
- Бесплатно для дома
- Платно для компаний: админка, аудит безопасности, бэкапы
- $50/юзер/мес
- 1000 компаний по 100 сотрудников = $60 млн/год ARR

### 3. Marketplace
- Комиссия 30% с платных модулей/приложений
- Дизайн-киты, оболочки, вертикальные решения
- CAC < $15 за счет сарафанного радио в среде гиков

### 4. Облачное хранилище
- Для тех, кто не хочет ставить свой Бэк-офис
- Конкуренция с iCloud/Google Drive: "Ваше облако, но данные у вас"

### 5. OEM-лицензии
- Китайские производители платят роялти
- Ставят CORE поверх Android/Linux со своими логотипами
- Готовые сервера, ноуты, планшеты

### 6. Обучение и сертификация
- Модель 1С: курсы для инженеров
- Сертификат "Core Architect"
- Пара денег за курс + поток кадров на рынок

## Go-to-Market: Троянский конь

### Phase 1: The Pro Tool
CORE как "Окружение для разработчиков и криэйторов":
- JS летает, P2P синхронизирует код/ассеты
- Слоган: "Твоё железо наконец-то проснулось"

### Phase 2: The Parasite
Поддержка легких оболочек для YouTube/Telegram/Spotify:
- "В CORE они не лагают и не жрут батарейку"
- Вертикальные дистрибутивы: CORE: Law, CORE: Construct, CORE: Health

### Phase 3: The Independence
"Core-Stick" или прошивка для дешевых планшетов:
- Полная замена хост-ОС на слабом железе
- Вторая жизнь старых ноутбуков

## Целевые сегменты

### B2B/B2G (первый приоритет)
**R&D в MedTech, FinTech, Оборонке:**
- Им нужна мобильность, но запрещено облако
- Сидят на тяжелых VDI (удаленные рабочие столы)
- CORE = Compliance-in-a-box: UX уровня Apple внутри закрытого контура
- Платят за безопасный доступ к IP (интеллектуальной собственности)
- $50/мес за юзера — копейки против штрафов за утечку

**Промышленность (Газпром и аналоги):**
- P2P-сеть на месторождениях без спутников
- Преддиктивная диагностика через Edge Computing
- Автономность от западных вендоров
- НИОКР-контракты как точка входа

**Юриспруденция, строительство, медицина, логистика:**
- CORE: Law — автоматический сбор дел с сайтов судов
- CORE: Construct — мгновенное открытие чертежей через WebGPU
- CORE: Health — доступ к картам пациентов по P2P
- CORE: Logistics — работа при падении интернета

### Массовый рынок
**Обычные юзеры:**
- "Бесконечное облако" за счет домашнего Бэк-офиса
- Умный поиск по всей цифровой жизни голосом
- Бесшовность между устройствами

**Гики/разработчики:**
- Кастомизация оболочек
- SDK на JS/TS
- "Битва оболочек" (как KDE vs GNOME)

**Дети/геймеры:**
- P2P Local Gaming (нулевой пинг)
- Бесконечная память для игр
- Моддинг на JS

## Стратегия "Мягкого входа" для масс

Не предлагать "переезд". Предложить "Теневого помощника":
1. CORE ставится за 2 минуты и сначала просто "наблюдает"
2. Через неделю: "Я уже скачал и разложил ваши файлы"
3. Пользователь не учился новому — он получил готовое

### Киллер-фича для входа
"Интеллектуальный поиск": сказать голосом "Найди фотку синего дивана, которую кидала жена в прошлом году" — система находит за 1 секунду среди 20 000 фото в разных чатах.

## Позиционирование против конкурентов

### Против Windows
- "Установка — это ад" → У нас: приложение = адрес
- "Зоопарк несовместимости" → У нас: один софт, любой экран
- "Тормоза из прошлого" → У нас: нет legacy-кода 90-х

### Против Apple
- "Железное рабство" → У нас: Apple-опыт на любом железе
- "Налог на память" → У нас: свой Бэк-офис, без подписок
- "Черный ящик" → У нас: кастомизация на JS

### Против ChromeOS
- ChromeOS = терминал к чужому облаку
- CORE = собственный суперкомпьютер, всегда с тобой

## Соглашение с "тройкой" (Яндекс/Сбер/ВК)

### Модель: Open Foundation
- Ядро — у независимого юрлица
- Тройка — соучредители, места в совете директоров
- Лицензия с правом fork под свои нужды
- Open Source для привлечения глобальных контрибьюторов

### Пилоты
- Сбер: платежный модуль вне зоны досягаемости Android
- Яндекс: офлайн-карты через P2P
- ВК: Mini Apps как нативный софт через WebGPU

### Защита от поглощения
Если один пытается выкупить долю другого — проект закрывается, код уходит в Open Source (MIT). Либо владеют вместе, либо не владеет никто.
